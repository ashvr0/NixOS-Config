# Overview
- Window manager: Hyprland
- Shell UI: quickshell
- Terminal: alacritty
- Shell: fish
- File manager: thunar
- Browser: librewolf
- Launcher: rofi
- Colors: matugen generated from wallpaper
- Font: JetBrains Mono Nerd Font
- 
## Keybinds
| Key | Action |
|---|---|
| `Super` + `T` | terminal |
| `Super` + E       | Open file manager |
| `Super` + W       | Open librewolf (browser)|
| `Super` + Super_L| Open app launcher |
| `Super` + Q | Close window        |
| `Super` + L | Lock screen         |
| `Super` + S | Screenshot |
| `Super` + ` | Wallpaper menu       |
